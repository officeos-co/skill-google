# Google Workspace

Search Google Drive and list upcoming Calendar events via a service-account key.

## Categories

- Productivity

## Credentials

| Key | Label | Kind |
|-----|-------|------|
| `service_account_json` | Service Account JSON | textarea |
| `calendar_id` | Calendar ID (optional) | text |

## Installation

This skill is included in [EnterpriseAgentOS](https://officeos.co) and requires no separate installation.
